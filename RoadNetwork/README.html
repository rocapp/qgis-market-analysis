Rocky Branches Plugin Suite

RoadNetwork, A plugin to help find the area of availability of a road network.
You'll need to set:
A roads layer, a start point, and a road distance to travel.

Installation:

  1. Go to Releases, and download the latest ZIP.
  2. Extract to C:\Users\YOURUSERNAME\.qgis2\python\plugins\PLUGINNAME
  • NOTE: Make sure you extract the folder such that the directory hierarchy is like this:
  • C:\Users\YOURUSERNAME\.qgis2\python\plugins\PLUGINNAME
  • NOT LIKE THIS:
  • C:\Users\YOURUSERNAME\.qgis2\python\plugins\PLUGINNAME\PLUGINNAME

Examples:

  • See the Examples directory for example qgis projects

Source Code:

  • All source code is under the respective plugin directory, e.g. RoadNetwork/
